REGRA OBRIGATÓRIA — Validação de setups:
Nunca usar imagens de um único objeto isolado (só cadeira, só monitor, etc) como exemplo de "setup".
Todo setup deve ter pelo menos 3 elementos funcionais combinados (ex: mesa + cadeira + notebook) e mostrar contexto de uso real.
Aplicar em galeria, cards, exemplos, kits, hero, categorias — qualquer lugar onde um "setup" é representado.
Ler a seção "Setup validation rules" no README.md para critérios completos.
