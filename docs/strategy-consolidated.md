## Riscos e mitigação

| Risco | Mitigação |
|---|---|
| Imagens bonitas mas inviáveis | Avisos claros + plano executável |
| Comoditização vs RoomGPT/InteriorAI | Foco em home office BR + ergonomia + produtos locais |
| UX complexa | Upload+estilo primeiro, refinamento depois |
| Recomendações comerciais agressivas | Separar essenciais, opcionais, patrocinados |
| Diagnóstico escondido | Mostrar no resultado + homepage |
| Baixa retenção pós-geração | Histórico + refinamentos + antes/depois |
